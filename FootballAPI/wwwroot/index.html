FootballAPI – Documentation

Overview of API endpoints for Athletes, Venues, Finance, and Image Upload.

Athlete Venue Finance Image Upload

Athlete Endpoints

CRUD operations for football athletes.

Athlete API endpoints
Method URL Description Request Body Response
GET /api/athlete Get all athletes None 200 OK + List<Athlete>
GET /api/athlete/{id} Get a single athlete by ID None 200 OK + Athlete / 404 Not Found
GET /api/athlete/unpurchased Get all unpurchased athletes None 200 OK + List<Athlete>
POST /api/athlete Create a new athlete JSON Athlete (without Id) 201 Created
PUT /api/athlete Update an existing athlete JSON Athlete (with Id) 204 No Content
DELETE /api/athlete/{id} Delete an athlete None 204 No Content / 404 Not Found

Venue Endpoints

CRUD operations for event venues.

Venue API endpoints
Method URL Description Request Body Response
GET /api/venue Get all venues None 200 OK + List<Venue>
GET /api/venue/{id} Get a single venue by ID None 200 OK + Venue / 404 Not Found
POST /api/venue Add a new venue JSON Venue (without Id) 201 Created
PUT /api/venue Update an existing venue JSON Venue (with Id) 204 No Content
DELETE /api/venue/{id} Delete a venue None 204 No Content / 404 Not Found

Finance Endpoints

Manage money, purchases and loans.

Finance API endpoints
Method URL Description Request Body Response
GET /api/finance Get current finance state None 200 OK + Finance object
POST /api/finance Create finance row (normally once) JSON Finance 201 Created
PUT /api/finance Update finance data JSON Finance 204 No Content
POST /api/finance/loan Take a loan (add money) { "loanAmount": number } 200 OK + updated Finance
POST /api/finance/buy Purchase an athlete (updates Athlete + Finance) { "athleteId": number } 200 OK + updated finance values
POST /api/finance/sell Sell a purchased athlete (remove from squad without deleting) { "athleteId": number } 200 OK + updated finance values

Image Upload

Upload images used for athletes or venues.

Image upload API endpoint
Method URL Description Request Body Response
POST /api/imageupload Upload a single image file FormData with "file" 200 OK + { "fileName": "example.jpg" }
